# career-portal
**How run the career-server**

```console
cd career-portal
```

activate virtual env

```console
pipenv shell
```

install requirements.

```console
pipenv install -r requirements.txt
```

run server 

```console
python3 manage.py runserver
```

open port http://127.0.0.1:8080
